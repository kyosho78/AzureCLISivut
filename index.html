Azure CLI Toiminnot

Mikä on Azure CLI?

Azure CLI on komentorivityökalu, jonka avulla voit hallita Azure-resursseja. Se tarjoaa helpon tavan automatisoida ja hallita Azure-palveluita komentoriviltä.

Yleisiä Azure CLI komentoja

  • az login - Kirjaudu sisään Azure-tilillesi.
  • az group create - Luo uusi resurssiryhmä.
  • az vm create - Luo uusi virtuaalikone.
  • az storage account create - Luo uusi tallennustili.
  • az webapp create - Luo uusi web-sovellus.
